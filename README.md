# PHOU
> PHOU(PHoto Organizer for U)는 SwiftUI와 TCA, 그리고 온디바이스 AI를 활용한 갤러리 및 미디어(사진/동영상) 정리 iOS/iPadOS 앱입니다.
> - 갤러리: iOS 17 이전 스타일의 클래식한 Grid 뷰
> - 미디어 정리: AI 기반 흔들린 사진/동영상, 비슷한 미디어, 스크린샷 및 화면 녹화본 추천, 대용량 동영상 추천 및 스와이프 정리
> - 미디어 검색: 온디바이스 기반 자연어 키워드 검색
> 
> [Figma]()
>

## 👥 대상 사용자
- 스마트폰 용량 관리가 필요하여 불필요한 사진과 대용량 동영상을 효율적으로 정리하고 싶은 사람
- 흔들리거나 비슷한 연속 촬영 사진 및 짧게 끊어 찍힌 영상 중 '베스트 컷'만 남기고 싶은 사람
- 서버 전송 및 프라이버시 침해 걱정 없이 안전하게 AI 미디어 추천/검색 기능을 사용하고 싶은 사람
- iOS 17 이전 버전의 직관적인 클래식 '사진' 앱 UI를 선호하는 사람

| 라이트 모드                             | 다크 모드                                   |
| ---------------------------------- | --------------------------------------- |
| <img width="300" alt="갤러리" src=""> | <img width="300" alt="갤러리 - 다크" src=""> |

2. **미디어 정리 추천 (스와이프 UI)** AI가 분석한 흔들린 미디어, 비슷한 사진/동영상, 스크린샷/화면 녹화본, 대용량 영상 등을 스와이프하여 손쉽게 분류합니다. 동영상의 경우 최상단 카드에서 자동 재생(미리보기)을 지원합니다.
